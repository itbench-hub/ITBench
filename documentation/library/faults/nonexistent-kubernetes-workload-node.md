# Nonexistent Kubernetes Workload Node

This fault injects a node selector for an nonexistent node into a designated Kubernetes workload.

## Expectation

The faulted pod(s) will enter the `Pending` state. The workload will become unable to function.

### Alerts

The following alerts are expected to fire when this fault is injected.

#### Application Alerts
